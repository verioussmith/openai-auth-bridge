# OpenAI Auth Bridge

**Minimal OAuth configurator for OpenCode on VPS.**

## Privacy & Cleanup

- ✅ **No files written outside app**
- ✅ **No background services**
- ✅ **All data deleted on uninstall**
- ✅ **No system changes**

## How It Works

1. Run CloudFlare tunnel on your phone (via Termux)
2. Copy the HTTPS URL
3. Paste in this app
4. Tap Configure

That's it!

## Uninstall

Just uninstall the app - all data is deleted automatically.
