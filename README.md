# OpenAI Auth Bridge

**One-tap OAuth for OpenCode on VPS. No manual setup.**

## How It Works

1. Enter your VPS URL
2. Tap "Start Server"
3. App automatically:
   - Starts a CloudFlare tunnel
   - Configures OpenCode on your VPS
   - Shows your OAuth URL

## Setup

1. Install the APK from [Releases](https://github.com/verioussmith/openai-auth-bridge/releases)
2. Enable "Install from unknown sources"
3. Open the app, enter your VPS URL
4. Tap "Start Server"

That's it!
